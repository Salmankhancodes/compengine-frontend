The Research Behind CompEngine

The vision of a comparison engine for data analysis has developed over many years starting with a paper in Royal Society Interface. The engine behind this website is based around our work on time-series analysis, which is described below.

The software

The engine underlying this website is based on software for performing highly comparative time-series analysis called hctsa (highly comparative time-series analysis). This Matlab-based software allows users to apply and compare the performance of thousands of scientific time-series analysis methods on their data and is freely-available for non-commercial use on github, and is accompanied by comprehensive online documentation on gitbook and a journal article.

The results

hctsa has been used for a wide range of real-world problems involving time-series data, spanning precision medicine to financial analytics. Successful applications include:

  • Classifying seizures from EEG brain recordings
  • Diagnosing Parkinson's disease from speech for precision medicine
  • Monitoring sleep-stage progression
  • Predicting schizophrenia from brain-imaging data
  • Forecasting catastrohpes in financial and ecological systems

For further examples of real-world application and commercial use of hctsa, see Engine Analytics.

How CompEngine works

Our research has suggested that real-world time series can be represented compactly using a small set of numbers, called features, that capture their important properties. When you upload time-series data to CompEngine, we compute these features from your data, and use them to find other time series, already in our library, that have similar feature values. We hope to improve this feature set over time to give the best possible matches.

The CompEngine feature set

The current set of 16 features was taken from our time-series analysis software package, hctsa , as a set that performs well on diverse time-series classification tasks and includes a diverse range of behaviors. The feature set used to retrieve neighbors in CompEngine can be grouped crudely as measuring: autocorrelation (CO_Embed2_Basic_tau_incircle_1, CO_Embed2_Basic_tau_incircle_2, AC_9, CO_FirstMin_ac, CO_trev_1_num, SB_MotifTwo_mean_hhh, PH_Walker_prop_01_sw_propcross) and predictability (FC_LocalSimple_mean1_taures, FC_LocalSimple_lfit_taures, EN_SampEn_5_03_sampen1), stationarity (SY_StdNthDer_1, SY_SpreadRandomLocal_50_100_meantaul, SY_SpreadRandomLocal_ac2_100_meantaul), distribution (DN_HistogramMode_10, DN_OutlierInclude_abs_001_mdrmd), and scaling (SC_FluctAnal_2_rsrangefit_50_1_logi_prop_r1).
